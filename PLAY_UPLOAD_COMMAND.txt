Fiber Locator Google Play Upload
================================

Yes, you need to update the native app in Google Play.

New app version:
versionCode 84
versionName 0.1.83

Run this on the DMI tablet in PowerShell:

scp linux@192.168.50.220:/home/linux/fiber.locator.dashboard/android-auto/app/build/outputs/bundle/release/app-release.aab "$env:USERPROFILE\Downloads\fiber-locator-0.1.83.aab"

Then upload this file to Google Play Console:

Downloads\fiber-locator-0.1.83.aab

SHA256:
6aafbcc40186f2f08b979603420343eec7258ad3b27c225327091e79851ab14e

Note:
Native Android phone map photo markers are easier to select, and the photo preview sheet now has left/right arrow controls for nearby photos in the current zoomed map area. The app still uses the existing lightweight map/photo loading path for smooth map performance.
